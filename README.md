# Jogo da Velha

Um jogo da velha (tic-tac-toe) interativo desenvolvido em **HTML**, **CSS** e **JavaScript** puro.

## 🎮 Características

- **Modo 2 Jogadores**: Jogue contra um amigo no mesmo dispositivo
- **Modo contra IA**: Desafie a inteligência artificial
- **Placar**: Acompanhe as vitórias de cada jogador
- **Interface Intuitiva**: Design simples e responsivo
- **Sem dependências externas**: Apenas vanilla JavaScript

## 🚀 Como Jogar

1. Escolha o modo de jogo:
   - **2 Jogadores**: Você e outro jogador alternam entre X (vermelho) e O (verde)
   - **Contra IA**: Você joga como X e a IA joga como O

2. Clique em qualquer casinha vazia do tabuleiro para fazer sua jogada

3. Forme uma linha horizontal, vertical ou diagonal com seus símbolos para vencer

4. O placar é atualizado automaticamente após cada partida

- A inteligência artificial utiliza uma estratégia de jogo para tentar vencer
- A IA escolhe suas jogadas de forma a maximizar as chances de vitória
- O jogo mantém o histórico de vitórias em um placar
